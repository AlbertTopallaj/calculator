# Calculator 

## Description
  The project consists of a website that presents functioning calculators. A user can register and login on the website. They can also upgrade their account to 'pro'. This gives access to certain calculators that are inaccessable otherwise.

## Purpose

- The purpose of this project is to gain knowledge about JavaScript and also add features that are based in JavaScript such as modals, login/signout system, render viewes based on states, mock payments and also shift between dark and light mode. The problem that this project solves is to make it easier and funnier to access a good calculator that meets the users needs. So this project is classified as for educational purposes only. 

## Instructions

- Step 1: Create a empty folder and name it however you want

- Step 2: Open the empty folder in any IDE you prefer

- Step 3: Open the terminal in your IDE

- Step 4: Copy this into your terminal:

### git clone https://github.com/AlbertTopallaj/calculator

- Step 5: Run with live server

- You have now succesfully downloaded a calculator onto your computer. Enjoy your calculator.

## Features

- Login
- Sign out
- Shift between Dark and Light mode
- Change the calculator
- "Pay" for the pro plan

## Additional Project Info

- The project is using Supabase as a database and for a few backend functions (fetch payload and proxy payment)
- The "payment" is mocked with beeceptor, which is an API mocking service. The payment is invoked and validated via supabase
- The functions therefore require an internet connection

## All the payments are not real and does not actually withdraw any money from any account. Its for educational purposes only.
